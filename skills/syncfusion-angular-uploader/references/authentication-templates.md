## JWT & Custom Headers

Pass authentication headers via `headers` in the request or by intercepting requests on the server. Use secure storage for tokens.

## Bearer Tokens

Attach `Authorization: Bearer <token>` for protected endpoints. Renew tokens on expiry and handle 401 responses.

## Item Templates

Customize uploaded item appearance using item templates—include thumbnails, file meta and action buttons.

## Progress Templates

Replace default progress UI with progress templates to show speed, remaining time or custom visuals.

## Styling Templates

Use CSS classes and Angular templates to ensure consistent design with app theme.
